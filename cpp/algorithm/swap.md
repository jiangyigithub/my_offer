## 算法基础操作
排序的基础操作是swap

```c++
//switch
int n1=1,n2=2;
int temp;
temp=n1;//backup n1
n1=n2;//override n1
n2=temp;//override n2
```

```c++
//walk togother
int n1=1,n2=2,n3=3;
int temp;
temp=n1;//backup n1
n1=n2;//override n1
n2=n3;
```
